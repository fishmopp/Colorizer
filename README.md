# Colorizer  
Colorizer originally designed for customised Kites. Works fine for all applications
that require custom colors: Bags, garments, ...

## Setup the example
  ```npm install```
  
## Start the example
  ```npm start```
  
The example application runs now under 127.0.0.1:8080/app/main.html
